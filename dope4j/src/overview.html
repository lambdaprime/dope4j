dope4j - provides access to Deep Object Pose Estimation (DOPE) models from Java. Based on Deep Java Library.

Usage

// cuboid model of ChocolatePudding which network was trained on
var objectCuboidModel = new Cuboid3D(new Point3D(),
  4.947199821472168, 2.9923000335693359, 8.3498001098632812);
var dopeService = new Dope4jFactory().createPoseEstimationService(
  "/models/ChocolatePudding.onnx", objectCuboidModel, cameraInfo);
System.out.println(dopeService.analyze(imageFile));
To create "ChocolatePudding.onnx" follow the instructions from Quickstart to isaac_ros_pose_estimation.
